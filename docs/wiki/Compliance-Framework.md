# Compliance Framework

EAIS automates the mapping of architectural designs to regulatory frameworks, significantly reducing the "Compliance Gap" during project inception.

## 🛡️ Supported Frameworks
- **Global**: ISO 27001, SOC2, NIST CSF
- **Financial**: PCI-DSS, SOX
- **Healthcare**: HIPAA, HITRUST
- **Privacy**: GDPR, CCPA

## 🔍 Assessment Features
- **Control Mapping**: Automatically links architectural components (e.g., encryption at rest) to specific framework requirements.
- **Gap Analysis**: Highlights missing security controls based on the selected requirements.
- **Evidence Ready**: Generates summaries that can be used directly in audit preparation.
- **Real-Time Scoring**: Provides a compliance percentage score for each architecture iteration.
